# App Modernization Branch

This branch (`app-modernization`) contains architecture documentation and diagrams for the Spring PetClinic Vets Service, created for application modernization planning purposes.

## Architecture Documentation

See [ARCHITECTURE.md](./ARCHITECTURE.md) for comprehensive documentation including:
- System architecture diagrams
- Component descriptions
- Database schema

## Purpose

This branch serves as a foundation for application modernization efforts, including:
- Understanding the current architecture
- Planning migration strategies
- Documenting modernization approaches
- Tracking modernization progress
